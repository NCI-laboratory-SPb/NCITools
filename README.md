# NCITools – Non-Covalent Interaction Analysis

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

NCITools provides two approaches to detect hydrogen bonds and estimate their energy:
- **Geometry‑based** – uses atomic coordinates and empirical thresholds.
- **Topology‑based (AIM/NCI)** – uses electron density from a cube file.
